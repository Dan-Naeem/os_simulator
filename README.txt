#Operating System Simulator 

This is a OS simulator, designed to carry out specific operations. 

It must manage processes and allow CPU access based off of a priority scheduling scheme w/ preemption.

It must request RAM size and page size from the user and set up the frame table accordingly. Sufficient space must be allocated for each process and pages must be added upon request. 

The number of hard drives is inputted by the user, and the OS must manage disk access requests.
